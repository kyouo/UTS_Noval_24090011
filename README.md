Deskripsi Singkat:
Proyek ini merupakan Dashboard Administrasi Produk sederhana yang dikembangkan menggunakan HTML, CSS, dan JavaScript murni (Vanilla JS). Tujuan dibuatnya aplikasi ini adalah menyediakan interface frontend yang modern dan responsif untuk melakukan pengelolaan data produk.

Fitur Utama:

Akses Login: Tersedia halaman login dengan validasi dasar sebelum masuk ke dashboard.

Ringkasan Data: Dashboard menampilkan kartu ringkasan (KPI) yang terisi otomatis, seperti Total Produk, Total Penjualan, serta Total Pendapatan yang diformat dalam Rupiah.

Manajemen Produk: Disediakan halaman khusus berupa tabel data produk yang terisi secara dinamis, lengkap dengan tombol simulasi untuk tindakan Edit dan Delete.

Tampilan & Desain: Menggunakan layout sidebar yang ringkas berbasis ikon, tipografi Poppins, serta desain bersih dengan memanfaatkan Flexbox.
